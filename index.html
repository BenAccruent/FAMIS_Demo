FAMIS 360 Demo

Responsive form with breakpoints set for:

  • 4 columns for desktop .col-md-4 (≥768px)
  • 2 columns for desktop .col-sm-6 (≥576px)
  • 1 column for mobile .col-12 (<576px)

Hiding a column/input will not break the grid and the columns/inputs after will reflow to fill the gap.

To see this in action: Toggle input Type 3 on/off

Responsive table with horizontal scrollbar (just on the table, not the whole page) based on viewport width.
